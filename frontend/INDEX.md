## ⚡ QUICK START (5 MINUTES)

### 1️⃣ Set Up Formspree (2 minutes)
```
1. Go to https://formspree.io
2. Create 2 forms: "Booking" & "Contact"
3. Copy the form IDs (format: f/abc123def456)
4. Save them somewhere
```

### 2️⃣ Update Form IDs (2 minutes)
```
File: pages/booking.html
Find: action="https://formspree.io/f/YOUR_FORMSPREE_ID"
Replace YOUR_FORMSPREE_ID with your booking form ID

File: pages/contact.html
Find: action="https://formspree.io/f/YOUR_FORMSPREE_ID"
Replace (2 places) with your contact form ID
```

### 3️⃣ Test & Deploy (1 minute)
```
1. Open index.html in browser
2. Test all pages and forms
3. Upload frontend folder to Netlify/hosting
4. Go live!
```
